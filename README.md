\# 🚗 Parking Management System



\## 📌 Project Overview



This is a full stack Parking Management System that helps manage vehicle entry, exit, billing, and reports.



\## 🚀 Features



\* Vehicle Entry System

\* Vehicle Exit \& Fee Calculation

\* Pay Bill using Vehicle Number

\* View Unpaid Bills

\* Active Vehicles Tracking

\* Daily Income Summary

\* Receipt Generation



\## 🛠️ Technologies Used



\### Frontend



\* React.js

\* HTML, CSS, JavaScript



\### Backend



\* Node.js / Python (based on your setup)

\* Express.js

\* MySQL



\## 📂 Project Structure



\* Backend → API \& database logic

\* Frontend → User interface



\## ▶️ How to Run



\### Backend:



cd Backend

python app.py   (or node server.js)



\### Frontend:



cd Frontend/frontend

npm install

npm start
